# Dashboard_DGCA
DGCA Aviation Safety & Incident Monitoring Dashboard

This project is an interactive analytical dashboard built using Plotly Dash, designed to support the Directorate General of Civil Aviation (DGCA), Government of India) in strengthening aviation safety oversight.

Incident Trend Analysis: Year-wise and month-wise patterns across all states and operators

Operator Insights: Incident frequency and category distribution for all airlines

Airport Insights: Airport-specific occurrence patterns and environmental factors

Geospatial Map: India map showing incident concentrations and severity markers

Advanced Filters: Flight phase, runway condition, visibility, ATA chapter, component/part, crew composition, and more

Responsive UI: Clean, modular layout optimized for multiple screen sizes
